# Day 13 – Student API using MVC Architecture

Today, I practiced building a simple Student API using the MVC (Model-View-Controller) architecture in Node.js and Express. Instead of keeping all the logic in a single file, the application was structured using controllers and routes for better organization.

I implemented basic API endpoints to manage student data including retrieving all students, adding a new student, and deleting a student. This exercise helped me understand how backend applications are structured in a modular way and how API logic is separated for better maintainability.
