# Backend Development – Day 27 Notes

## Sorting and Searching in APIs

### Search
Used to find specific data using query parameters.

Example:
?search=laptop

### Sorting
Used to arrange data in order.

Ascending → ?sort=asc  
Descending → ?sort=desc  

## Benefits

- Improves user experience
- Helps find data quickly
- Efficient data handling

## Learning Outcome

Today I learned how to implement sorting and searching in backend APIs using query parameters.
