# Day 24 – Backend Deployment Basics

Today, I learned about backend deployment and how applications are made available on the internet for users. I explored the basic steps involved in deploying an application, including preparing the project, configuring environment variables, choosing a hosting platform, and running the application on a server. I also understood the difference between local servers and remote servers.

From today’s learning, I gained a clear understanding of how backend applications are deployed in real-world environments. I learned the importance of deployment in making applications accessible to users and how environment variables are used to store sensitive data securely. This session helped me understand how development transitions into real-world usage.
