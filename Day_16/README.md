# Day 16 – MongoDB Fundamentals

Today, I learned the fundamentals of MongoDB and how it is used as a NoSQL database in modern web applications. I understood how MongoDB stores data in the form of collections and documents instead of traditional tables and rows. I also explored how its flexible schema allows developers to store structured data in JSON-like format, making it easier to manage dynamic data in applications.

From today’s learning, I gained a clearer understanding of how MongoDB works and how it differs from relational databases. I learned the basic concepts of inserting, retrieving, and organizing data in collections. This helped me build a strong foundation in database management and improved my understanding of how backend applications store and handle data efficiently.
