# Day 11 – Building RESTful APIs with Node.js & Express

Today, I learned about RESTful APIs and how they enable communication between frontend and backend applications. I explored how to create APIs using Node.js and Express.js and practiced implementing endpoints such as GET, POST, PUT, and DELETE to manage user data.

I also understood the structure of API requests including URL, method, headers, and request body. Through this learning, I was able to build a simple REST API and test it using Postman.
