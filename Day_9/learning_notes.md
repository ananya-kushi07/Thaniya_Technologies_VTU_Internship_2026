# Backend Development – Day 9 Notes

## What is CRUD?

CRUD represents the four basic operations used in database management.

C – Create new data  
R – Read existing data  
U – Update existing data  
D – Delete data  

These operations form the foundation of most backend APIs.

## REST API Methods

POST → Create data  
GET → Retrieve data  
PUT → Update data  
DELETE → Remove data  

## CRUD Flow in Backend

Client Request → API Route → Controller Logic → Database Operation → Response to Client

## Example Operations

Create User → POST /users  
Get All Users → GET /users  
Update User → PUT /users/:id  
Delete User → DELETE /users/:id
