# Day 9 – REST API Development with MongoDB

On Day 9, I focused on implementing CRUD (Create, Read, Update, Delete) operations using MongoDB and Mongoose in a Node.js backend application.

I learned how backend servers handle API requests to create new records, retrieve stored data, update existing entries, and delete records from the database. Using Express.js along with Mongoose made it easier to define routes and interact with MongoDB collections.

This helped me understand how real-world backend systems manage and manipulate data through RESTful APIs.
