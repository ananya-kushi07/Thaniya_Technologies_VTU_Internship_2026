# Day 8 – MongoDB and Mongoose Integration

Today, I started learning about MongoDB and Mongoose and how they are used to manage data in backend applications. MongoDB is a NoSQL database that stores data in the form of collections and documents, which allows flexible and scalable data management.

I also learned how Mongoose acts as an Object Data Modeling (ODM) library for Node.js, helping developers define schemas and models to interact with MongoDB more efficiently.

During this session, I practiced creating a simple schema and model and understood how to connect a Node.js application to the MongoDB database.
