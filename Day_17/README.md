# Day 17 – MongoDB CRUD Operations

Today, I learned how to perform CRUD (Create, Read, Update, Delete) operations using MongoDB with Express.js. I practiced connecting a Node.js application to MongoDB using Mongoose and created API endpoints to manage product data. This included adding new products, retrieving stored products, updating existing data, and deleting records from the database.

Through this implementation, I gained a deeper understanding of how backend applications interact with databases to manage application data. Practicing these operations helped me understand how MongoDB queries work and how APIs handle database communication in real-world backend systems.
